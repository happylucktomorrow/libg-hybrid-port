# Symbol Classification

This directory contains classified symbol lists generated by `scripts/classify_symbols.py`.

## Files

| File | Description |
|------|-------------|
| `game_logic_symbols.txt` | Symbols to port to clean C++ |
| `internal_symbols.txt` | Symbols to keep as original ARM blob |
| `unclassified_symbols.txt` | Symbols that didn't match any pattern |

## Classification Criteria

**Game logic** (port to C++) includes:
- `Logic*` classes (commands, components, data, managers)
- Protocol messages (`*Message`)
- Protocol codec and serialization
- Game state managers
- Avatar/player management
- JNI entry points

**Internal** (keep as ARM) includes:
- Display framework (`DisplayObject`, `MovieClip`, `Sprite`, `TextField`, `Stage`)
- Renderer (`IsoCamera`, `TileEngine`, `HomeMapRenderer`, `Shader*`)
- UI screens and menus (`*Screen`, `*Menu`, `*Popover`, `*HUD`)
- Audio (`SoundManager`)
- Animation and tweening
- Particle systems
- SC bundle decoding
- Third-party code (ICU, TweetNaCl)
- C++ runtime (STL, `__cxa_*`)

## Regeneration

```bash
python3 ../scripts/classify_symbols.py ../original/libg.so
```
